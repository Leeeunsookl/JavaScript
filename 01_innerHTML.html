01. innerHTML

        Element.prototype.innerHTML 프로퍼티는 getter, setter 모두 존재하는 프로퍼티다.
        요소 노드의 html 마크업을 취득하거나 변경한다.
        textContent 프로퍼티는 html 마크업을 무시하고 텍스트만 반환하지만,
        innerHTML 프로퍼티는 HTML 마크업이 포함된 문자열을 그대로 반환한다.
태그 엘리먼트의 값을 읽거나, 변경할 때 innerHTML속성을 사용한다.
        innerHTML 프로퍼티를 사용하면 DOM 조작이 간단하고 직관적이라는 장점이 있다.
        사용자한테 입력받은 데이터를 그대로 innerHTML에 할당하는 것은 보완에 취약하다.
        악성 코드를 html 마크업 내에 포함한다면 파싱(번역) 과정에서 그대로 실행될 수 있다.
